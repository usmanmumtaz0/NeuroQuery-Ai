# NeuroQuery AI

**NeuroQuery AI** is an intelligent AI Research Assistant that routes queries to specialized expert personas. Unlike generic chatbots, NeuroQuery AI analyzes user intent and delivers precise responses from experts in **Technology, Finance, and News**, all powered by Google’s **Gemini LLM**.

---

## Key Features

- **Intelligent Query Routing:** Dynamically classifies queries and sends them to the most relevant expert (Tech, Finance, or News).  
- **Multi-Chain Architecture:** Built using LangChain with ChatPromptTemplate and StrOutputParser for accurate and structured responses.  
- **State Management:** Session-based conversation history and metrics tracking for a smooth chat experience.  
- **Summary Memory:** Option to summarize the ongoing conversation for quick review.  
- **Modern UI/UX:** SaaS-style interface built on Streamlit with chat bubbles, animations, and loading spinners.  
- **Gemini LLM Powered:** High-speed, accurate reasoning using Google’s Gemini 2.5/3 Flash model.
